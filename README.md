# 使用指南

## 文件列表

- `/data/mirrors.json`：维护主页镜像列表的源文件
- `/data/task_status.json`：镜像工具`mirror`生成的镜像同步状态文件

## 操作列表

本地仓库执行 `git push origin master` 代码推送即可。
